Technical Guide

11. Skills 源码：Skill 是怎么安装、解析和约束工具的

理解 installer、parser、validation、permissions、security_scanner 和 tool_policy。

Skill 源码入口

backend/packages/harness/deerflow/skills/
├── installer.py
├── parser.py
├── permissions.py
├── security_scanner.py
├── slash.py
├── storage/
├── tool_policy.py
├── types.py
└── validation.py

关键职责

  • parser.py：解析 Skill 内容；
  • installer.py：安装 Skill；
  • validation.py：校验 Skill 结构；
  • permissions.py：处理权限；
  • security_scanner.py：检查风险；
  • tool_policy.py：约束工具使用；
  • storage/：Skill 存储抽象。

调用链

Skill 文件 / archive
  ↓
installer 安装
  ↓
parser 解析
  ↓
validation 校验
  ↓
permissions / security scanner
  ↓
tool policy / prompt 注入
  ↓
Agent 执行时受影响

完成标准

你知道 Skill 源码不只是读 Markdown，而是一套安装、解析、校验、权限、安全和工具策略系统。
