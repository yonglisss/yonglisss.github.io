DeerFlow 教程

  • DeerFlow 教程总览 从跑起来到做实战，系统学习 DeerFlow 2.0 的本地部署、配置、架构和能力扩展。
  • 快速开始：先把 DeerFlow 跑起来 用最短路径完成 DeerFlow 2.0 的本地配置、环境检查、Docker / 本地启动和第一个任务验证。
  • 配置文件：看懂 config.yaml 拆解 DeerFlow 2.0 的核心配置：模型、API key、搜索、沙盒、运行目录、skills 和渠道入口。
  • 核心概念：DeerFlow 到底在调度什么 理解 DeerFlow 2.0 的核心概念：super agent harness、tools、skills、sub-agents、memory、sandbox 和 context engineering。
  • 项目结构：从目录看懂 DeerFlow 通过 DeerFlow 2.0 的仓库目录理解前端、Gateway、Agent Runtime、Sandbox、Skills、MCP 和配置系统的分工。
  • 一次任务的完整执行链路 从用户输入到最终结果，拆解 DeerFlow 2.0 中 Gateway、Agent Runtime、模型、工具、沙盒、子 Agent 和文件产物的协作过程。
  • 实战案例 DeerFlow 的案例文档占位，后续用于放可复现的技术教程。
