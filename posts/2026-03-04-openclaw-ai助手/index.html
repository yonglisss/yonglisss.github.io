📌 工具介绍

OpenClaw AI助手 是一个 AI助手平台，主要用于 提升工作效率，自动化重复任务。

核心功能：

  • 智能对话和问答
  • 自动化任务执行
  • 多平台集成
  • 个性化配置

适用场景：

  • 日常任务自动化
  • 信息查询和整理
  • 多平台消息管理
  • 个性化提醒和通知

⚡ 快速上手

安装

# 使用npm安装
npm install -g @openclaw/openclaw

# 或者使用Docker
docker run -it openclaw/openclaw

Hello World

# 启动OpenClaw
openclaw gateway start

# 连接到助手
openclaw agent --message "你好，我是元英"
